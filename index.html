ReWright: Real-time Comment Detoxification in Developer Community

Duration: 2024 - Present

Role: Thesis Supervisor

Team Size: 5 researchers

Institution: BRAC University

Abstract

ReWright leverages Large Language Models (LLMs) and ToxiCR's framework to rephrase toxic code review comments in real-time, offering polite and constructive alternatives. This tool aims to foster respectful communication and improve collaboration in developer communities. This project combines the strengths of advanced toxicity detection with contextual paraphrasing, ensuring comments remain technically accurate while eliminating offensive language.

Research Goals
  • Goal 1:Develop a robust framework for detecting toxic comments during code reviews using ToxiCR’s dataset and classification methodology.
  • Goal 2:Design and integrate a real-time paraphrasing mechanism powered by LLMs to rephrase offensive comments while preserving technical context.
  • Goal 3:Implement a user-friendly interface for developers to interact with the tool, allowing for easy integration into code review workflows.
  • Goal 4:Assess the impact of the tool on developer productivity, collaboration, and overall code quality.
Technologies
  • Open-source Large Language Models
  • Natural Language Processing
  • ToxiCR Framework
  • Real-time Processing Systems
Expected Impact

ReWright aims to transform developer communication by promoting respectful discourse, reducing toxicity in code reviews, and fostering a more inclusive and collaborative development environment.

Prioritizing Test Cases for Mutation Testing using LLM

Duration: 2024 - Present

Role: Thesis Supervisor

Team Size: 5 researchers

Institution: BRAC University

Abstract

This project focuses on improving the efficiency of software mutation testing, a critical process for detecting software errors, which currently suffers from high computational costs and limited test suite coverage. By adopting a hybrid approach, the research combines LLM-generated test cases with traditional methods to prioritize and optimize test case execution. The aim is to identify effective test case generation strategies and integrate them into prioritization techniques for a more efficient mutation testing model.

Research Goals
  • Develop an efficient strategy for generating test cases using Large Language Models (LLMs)
  • Design and implement a prioritization technique to determine the most impactful test cases
  • Evaluate the effectiveness of the hybrid approach using algorithms and statistical analyses
Methodology
  • Research Approach: Hybrid testing approach integrating LLM-generated test cases with traditional mutation testing techniques
  • Data Collection: Analysis of software testing datasets and outputs from both LLM-generated and traditional test cases
  • Analysis: Statistical techniques and algorithms to measure effectiveness and evaluate prioritization strategies
Expected Impact

The hybrid approach is expected to enhance the efficiency and coverage of software mutation testing, reducing computational demands while improving error detection rates. By integrating LLM-generated test cases and prioritization techniques, this research could redefine testing strategies and pave the way for more scalable, reliable, and cost-effective software testing methodologies in the industry.

Conference Paper 2022 Published

Improving Action Quality Assessment Using Weighted Aggregation

Farabi, S., Himel, H., Md. Fakhruddin Gazzali Fahim, Hasan, M.B., Kabir, M.H., Farazi, M.

IbPRIA 2022. | Vol 13256. Springer, Cham.

Abstract

Action quality assessment (AQA) aims at automatically judging human action based on a video of the said action and assigning a performance score to it. The majority of works in the existing literature on AQA divide RGB videos into short clips, transform these clips to higher-level representations using Convolutional 3D (C3D) networks, and aggregate them through averaging. These higher-level representations are used to perform AQA. We find that the current clip level feature aggregation technique of averaging is insufficient to capture the relative importance of clip level features. In this work, we propose a learning-based weighted-averaging technique. Using this technique, better performance can be obtained without sacrificing too much computational resources. We call this technique Weight-Decider(WD). We also experiment with ResNets for learning better representations for action quality assessment. We assess the effects of the depth and input clip size of the convolutional neural network on the quality of action score predictions. We achieve a new state-of-the-art Spearman’s rank correlation of 0.9315 (an increase of 0.45%) on the MTL-AQA dataset using a 34 layer (2+1)D ResNet with the capability of processing 32 frame clips, with WD aggregation.
