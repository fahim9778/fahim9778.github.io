LLM-Based Software Engineering
LLM Integration Software Engineering Development Tools

This research explores the transformative role of Large Language Models (LLMs) in software engineering, focusing on their ability to enhance development efficiency, code quality, and project management. By evaluating tools like Llama, OpenAI's GPT models, and Google's BERT, the study aims to assess LLMs' accuracy and usability. Using mixed methods, it combines literature reviews and developer surveys to analyze the impact of LLMs on software practices, with future research focusing on performance optimization, tool customization, and integration with development environments.

Key Research Aspects:
  • Evaluation of LLM tools in software development
  • Impact assessment on development efficiency
  • Code quality improvement analysis
  • Project management integration
Automated Code Generation Systems
AI Code Generation Automated Development Software Automation

I am passionate about the potential of automated code generation, investigating how AI can be harnessed to improve coding efficiency and reduce manual efforts. My research explores creating intelligent systems that can generate code snippets, entire modules, or even bug fixes autonomously, streamlining the software development process and enhancing developer productivity.

Key Research Aspects:
  • AI-powered code generation techniques
  • Automated bug fixing systems
  • Code completion and suggestion tools
  • Development workflow automation
LLMs in Computer Science Education and Societal Expectances
CS Education AI Ethics Learning Technology

I explore how Large Language Models (LLMs) can revolutionize computer science education and shape our society's future. My research focuses on developing LLMs as intelligent tutoring companions that can adapt to each student's learning style, help detect academic misconduct, and foster meaningful discussions about AI's role in education.

What fascinates me most is how these AI tools challenge our traditional views of teaching and learning. As we partner with AI in education, we're not just creating better learning tools – we're reimagining what it means to learn, create, and grow in a world where human and artificial intelligence work together.

Key Research Aspects:
  • LLMs as educational companions
  • AI-assisted plagiarism detection
  • Ethics and philosophy of AI in education
  • Human-AI collaboration in learning
